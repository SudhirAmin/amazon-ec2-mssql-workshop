## Storage performance

To synthesize a load on the volume, use a tool such as [Diskspd Utility](https://gallery.technet.microsoft.com/DiskSpd-A-Robust-Storage-6ef84e62). 
Connect to one of the cluster nodes with RDP and run the following with the Diskspd tool.

Test the data disk (GP2 Striping)

```bash
diskspd.exe -d60 -b4k -o1024 -t32 -L -Sh -r -w50 -W60 -c500G D:\test.dat
```

Test the Local NVMe

```bash
diskspd.exe -d60 -b4k -o1024 -t32 -L -Sh -r -w50 -W60 -c100G E:\test.dat
```
